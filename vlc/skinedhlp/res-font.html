Font

A Font resource loads a font file so that it can be used by items in the skin.

All major font formats (TrueType & OpenType) are supported by VLC, but the Skin Editor can only show a preview of TrueType-Fonts (*.ttf).

Attributes

id
The name identifying this Font. The skin's items use this ID to refer to that font.
This ID has to be unique and is required to be set.

file
The path to the font file relative to the location of the skin file. This attribute is required to be set.

size
The font size in points. This is by default 12.
